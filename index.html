Biografía

Hola, soy María González, desarrolladora web full-stack con más de 5 años de experiencia en la creación de aplicaciones web modernas y escalables. Me apasiona el desarrollo frontend y disfruto trabajando con tecnologías como HTML5, CSS3, JavaScript y frameworks modernos como React y Vue.js.

Durante mi carrera profesional, he trabajado en diversos proyectos que van desde sitios web corporativos hasta aplicaciones empresariales complejas. Me especializo en crear interfaces de usuario intuitivas y accesibles, siempre siguiendo las mejores prácticas de desarrollo y diseño web. Además, soy una firme defensora del código limpio y la arquitectura de software bien estructurada.
